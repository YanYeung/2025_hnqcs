# 比赛成绩统计系统 (Competition Scoring System)

## 1. 项目简介

**比赛成绩统计系统** 是一款专为各类竞赛（特别是青少年创新实践大赛、机器人比赛等）设计的轻量级 Web 应用。它旨在解决传统纸质或 Excel 记分效率低、排名统计滞后、现场展示不直观的问题。

该系统基于现代前端技术构建，支持**多赛项管理**、**多角色协作**（管理员/裁判员）、**实时排名大屏展示**以及**自动化奖项计算**。数据存储在本地浏览器中，部署简单，即开即用。

### 核心亮点

* **多赛项支持**：单系统可同时管理多个不同类型的比赛项目，数据互不干扰。
* **角色权限分级**：管理员掌控全局设置，裁判员仅关注负责赛项的成绩录入。
* **实时大屏**：独立的展示端视图，适配大屏幕，实时刷新排名和获奖情况。
* **智能辅助**：支持 Excel 名单导入、自动匹配选手信息、自动计算“两轮取最佳”成绩。
* **奖项自动计算**：根据预设比例（一、二、三等奖）自动在导出数据中通过算法生成获奖等级。

## 3. 使用说明

* * *

### 3.1. 初始化设置（管理员）

首次使用系统时，请以管理员身份登录。

* **默认账号**：admin
  
* **默认密码**：admin （登录后建议立即修改）
  

#### 步骤一：配置赛事结构

1. 点击右上角的 **⚙️ (设置)** 按钮。
  
2. **基本设置**：修改比赛主标题，并设置**获奖比例**（例如：一等奖15%，二等奖25%，三等奖30%）。
  
3. **赛项管理**：添加本次比赛的所有分赛项（如“无人机障碍赛”、“机器人搬运赛”等）。
  
4. **裁判员管理**：为每个赛项创建裁判员账号，系统会自动将裁判锁定在指定赛项下。
  

#### 步骤二：导入参赛名单

1. 在主界面选择对应的赛项。
  
2. 点击 **“下载模板”** 获取 Excel 模板文件。
  
3. 按格式填入选手编号、姓名、组别（初级组/高级组）。
  
4. 点击 **“导入名单”** 将选手数据载入系统。
  

* * *

### 3.2. 成绩录入（裁判员/管理员）

裁判员使用管理员分配的账号登录，登录后将直接进入其负责的赛项界面。

1. **输入编号**：在录入框输入选手编号（支持模糊搜索），系统会自动填充姓名和组别。
  
2. **选择轮次**：选择当前是“第一轮”还是“第二轮”。
  
3. **录入数据**：输入“得分”和“耗时”。
  
4. **提交**：点击提交。
  
  * 注：下方“最近录入”列表可实时查看历史记录，点击铅笔图标可修正错误数据。

* * *

### 3.3. 排名与奖项（查看与导出）

系统采用**“分数优先，时间次之”**的排名逻辑，且自动取两轮中表现最好的一轮作为最终成绩。

1. **查看排名**：右侧（移动端为下方）排行榜会实时更新。
  
2. **大屏展示**：
  
  * 点击顶部导航栏的 **“🖥️ 打开大屏”** 按钮。
    
  * 该页面为纯净版设计，适合投影到现场大屏幕，支持通过下拉菜单切换不同赛项的排名。
    
3. **导出成绩**：
  
  * 在排行榜上方点击 **“导出”** 按钮。
    
  * 下载的 CSV 表格中包含：排名、选手信息、各轮次详情、最终成绩以及**根据比例自动计算的获奖等级**。
    

* * *

### 3.4. 注意事项

1. **数据存储**：本系统使用浏览器本地存储 (LocalStorage)。**请勿在比赛过程中清除浏览器缓存**，否则会导致数据丢失。
  
2. **数据备份**：建议定时点击“导出”按钮下载 CSV 表格作为数据备份。
  
3. **多设备协作**：由于是纯前端应用，数据存在于当前电脑中。若需多台电脑协作（如A电脑录入，B电脑大屏），目前版本需在同一台电脑上操作或连接扩展显示器；或通过部署服务器版本（需后端支持）实现。
